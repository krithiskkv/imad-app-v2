Technical Features

This webapp is built on the IMAD console, an nginx server acting as reverse proxy

Front End

The front end is built with W3 CSS templates using HTML CSS and jQuery. The site is responsive and works well with devices of all sizes. Different templates are used for the home page and blog pages

Back End

The server side code is built with JavaScript, Express.js and node.js. express-session is used for session control and crypto module is used for hashing passwords

Database

The database used is PostGRESql. The PostGres tables store blog data such as the content of the blog, comments and likes. Postgres tables are normalised to prevent data redundancy

Additional Features

This website is SQL Injection proof and XSS attacks proof! The code follows DRY principle - the blog drop downs and blog post pages are generated dynamically without repeating any code!
